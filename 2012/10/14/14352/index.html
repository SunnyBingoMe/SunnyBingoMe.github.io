瑞典语 语法： 形容词->En/Ett/Pl词尾变化总结 [Svenska] [Adjectiv]->[En/Ett/Plural/O-/Bestämdform]

Leave a reply

OBS：顺序：En词前 -> Ett词前 -> Plural词前。

 

一、原形 / EN词前 -> ETT单.

1. BRA -> BRA.

bra -> bra

 

2. 元音 -> +TT

t.ex:

ny -> nytt

blå -> blått

 

3.1 ~RD -> ~RT; ~ND -> ~NT

t.ex:

hård -> hårt

känd -> känt

 

3.2 ~D (非RD, ND) -> ~TT

t.ex:

god -> gott

 

4. ~EN -> ~ET

t.ex:
liten -> litet
vaken -> vaket

 

5. ~T -> 不变 (~T)

t.ex:

intressant -> intressant

 

6. 一般 -> +T

t.ex:

dyr -> dyrt

modern -> modernt

fin -> fint

stor -> stort

 

二、原形 / EN词前 -> 复数.

1. BRA -> BRA.

bra -> bra

 

2. LITEN -> SMÅ

liten -> små

 

3. Gammal //辅音前的m不再双写

gammal -> gammalt ->gamla

 

4. 211规则: ~ER, EL, EN

去 e，再按“一般”+a。

t.ex:

vacker -> vackert -> vackra

enkel -> enkelt -> enkla

vaken -> vaket -> vakna

 

4. 一般 -> +A

t.ex:

trevlig -> trevligt -> trevliga

trött -> trött -> trötta

ny -> nytt -> nya

god -> gott -> goda

känd -> känt -> kända

额外变化：结尾m双写

t.ex:

dum -> dumt -> dumma

 

三、Bestämdform.

1. 两个个别: LITEN, BRA

liten 单数 Best-form: lilla, 复数 和 “一般情况”相同.

bra ->bra.

 

2. 一般情况:

不分en/ett, 不分单复数, 都照抄 OBest form 的复数 形式.
