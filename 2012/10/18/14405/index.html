瑞典语 语法: 动词时态 词尾变化总结 [Verb] [Presens/Preteritum/Infinitiv/Futurum]

Leave a reply

注：此文中的顺序：Presens -> Pret`e:ritum -> Infinitiv (ska + inf = Futurum)。

 

Gr.1 ~AR -> ~ADE-> ~A

t.ex: tittar -> tittade -> titta,

betalar, studerar, jobbar, parkerar, arbetar, städar ≈ dammar, badar ((bathe, swim)) , dusch:ar/²d`uʃ:ar/ ((shower)), talar, pratar, frågar ≠ svarar, stannar ((stop)), lämnar ≠ hämtar, börjar ≠ slutar, hälsar ((salute)), spelar, cyklar, bakar, diskar, tvättar, lagar, handlar, öppnar, vaknar (sluta sova).

 

Gr.2 ~ER

 

Gr.2.B ~P/T/K/S/X–ER -> ~P/T/K/S/X–TE -> ~P/T/K/S/X–A

t.ex: söker ((look for)) -> sökte -> söka,

blåser ((blow)), hjälp-er, klipper, köper, leker (=spelar), låser /l’å:ser/, läser, märker ((mark, label)), res-er, rök-er, släck:er ((put out [fire])), ste:ker ((roast)), sänker  ((≠ höjer)), tycker = tänker (≈tror ((believe)) ), åker, gifter, byt-er ((change, exchange)), möter (=träffar), sköter (ta hand om), värker /v`är:ker/ ((ache, hurt)), väcker /v`ek:er/((wake [someone up])).

 

Gr.2.C ~D–ER -> ~D–E -> ~D–A

//即: 相比”其他一般情况”, 省略一个d (不双写).

t.ex: använder -> använde -> använda, 

händer ((happen)), tänder (få att brinna).

Gr.2.A 其他一般情况:~ER(非 ~P/T/K/S/X,D) -> ~DE -> ~A

t.ex: behöver -> behövde -> behöva,

anmäler ((report)), bygger, fyller ((fill)), följ–er ((follow)), gäller ((be valid, apply)), höjer (≠ sänker), ring-er, slänger ((throw)), stänger ((close)), ställer (—>står ((stand)) ), svänger ((swing)), väger ((weigh)), äger.

 

Gr.2.A.额外: MM, NN在辅音前失去双写.

t.ex:

känner (vet) -> kände -> känna.

glömmer -> glömde  -> glömma.

drömmer -> drömde -> drömma.

bestämmer ((determine)), gömmer ((hide)), tömmer ((empty)).

 

Gr.3 Korta Verb 短动词: ~R -> ~DDE -> ~(去R)

//verbet, verb, verben

t.ex: bor -> bodde -> bo,

flyr ((flee, escape)), ror /ro:r/(ror [båten]) (vs. rör Gr4.2F) , tror, beror /ber`o:r/((depend)), glo:r ((stare)), sno:r ((twist)), bre:r ((spread)), sker /ʃe:r/(=händer), avskyr /²`a:vʃy:/((hate)), spyr ((vomit)), syr, mår, når ((reach)), spår ((predict)), sår ((sow, plant)), klär ((dress)).
