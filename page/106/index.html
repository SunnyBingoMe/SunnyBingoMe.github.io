Ubuntu10.10 中国定制版下载

Leave a reply

感谢绝版黑色的投递
早在2010年10月份的时候Canonical就表示，Ubuntu中国定制版最晚将在2011年4月份推出。日前，Ubuntu创始人Mark Shuttleworth造访北京，与我国开发者进行一些关于创新方面的互动沟通，并且与中国的OEM合作伙伴进行必要的交流。Shuttleworth还带来了一个好消息，那就是Ubuntu的中国版产品已经正式发布了，用户可以直接从网站下载使用，而且产品CD也正在运往中国的途中。Shuttleworth指出：“这是第一个中国版本产品，Ubuntu也会持续地完善其性能与质量，希望中国用户会发现Ubuntu产品更多的优点。”

实际上，中国版与国际版的区别只是内置组件的不同，以及语言的默认设置为中文，在功能方面并没有什么重大改动。

Ubuntu中国版下载地址：
http://china-images.ubuntu.com/

Ubuntu 10.10中国版下载地址：
http://china-images.ubuntu.com/10.10/daily-live/current/maverick-desktop-i386.iso

Ubuntu 11.04中国版（测试版）下载页面：
http://china-images.ubuntu.com/11.04/daily-live/current/

    20个替代电脑软件的优秀在线工具（上）

    你好，亲爱地读者朋友! 如果你是第一次来到这里，你也许可以尝试订阅 RSS feed 关注本博客的新文章.

    这是一篇翻译的文章，英文好的同学请直接看原文。  由于原文较长，图片较多，分为两篇文章发布。

    电脑软件给我们提供了大量的帮助，作为我们工作、设计、爱好和兴趣的好帮手，然而，这些软件都有被在线工具取代的趋势，尽管这些在线工具没有桌面软件功能强大，但却更加方便易用和具有扩展性，更重要的是，你不需要进行任何安装，只需要打开网页访问这些在线工具就可以了。 本文将介绍20个替代桌面软件的优秀在线应用程序，希望你会喜欢。

    一、Google Docs

    Google Docs（google 文件）是一个让你在线创建文档、电子表格等文本应用的工具。（不幸的是，国内需要翻墙） 用来替代以下软件：

    Microsoft Office

    Open Office

    二、Meebo

    Meebo是一个在线聊天工具，可以同时登陆你的一个或多个你的IM账户。（支持中文） 用来替代以下软件：

    Facebook Chat

    Yahoo Messenger

    AIM

    Windows Live Messenger

    三、Slide Rocket

    幻灯展示广泛应用于商业、教育和项目中，Slide Rocket则是一个在线制作幻灯片的工具，他提供了绝大多数PowerPoint所具备的功能。 用来替代以下软件：

    Microsoft PowerPoint

    OpenOffice Impress

    四、Aviary

    Aviary是一个强大的提供在线图片编辑、logo制作、网页模板制作和音频编辑等功能的应用.

    用来替代以下软件：

    Adobe Photoshop

    Adobe Illustrator

    Soundbooth

    五、Mint

    你觉得没有时间来管理你的财务，那么Mint正是你所需要的，他是一个免费的在线个人财务管理系统，帮助你建立个人账户，进行预算设置和个人储蓄管理等。

    用来替代以下软件：

    Quicken（这个貌似用的不多？）

    六、JayCut

    JayCut是一个在线视频编辑应用，良好的用户界面让你方便的添加和编辑视频并输出到youtube或者本地。

    用来替代以下软件：

    Windows Movie Maker

    Ulead

    其他视频编辑软件

    七、WobZip

    Wobzip是一个在线解压缩工具，提供ZIP, RAR和其他格式的解压，同时还能在解压缩之前预先进行文件扫描预览。

    用来替代以下软件：

    各种解压缩工具

    八、Zamzar

    Zamzar是一个功能强大的在线文件转换工具。他可以转换各种形式的文件：如文档、图片、音乐、电子书、压缩文件和CAD文件等，只需要按照4个简单的步骤就可以轻松完成转换。

    用来替代以下软件：

    文件转换工具

    九、 Hulu

    Hulu是一个让你在电脑里面看电视和视频节目的网站，如果你错过了你喜欢的节目，可以在这里重新观看。Hulu仅对美国用户开放，如果你不在美国，却又想访问它，可以点击这里。

    用来替代以下软件：

    电视节目转换软件

    十、TokBox

    如果你喜欢视频聊天，那么TokBox是一个很好的在线视频聊天应用，作为免费用户，你可以和最多20个好友进行聊天，简单注册之后，邀请你的好友加入然后就可以开始视频聊天了。

    用来替代以下软件：

    视频聊天软件
