瑞典语 语法： 形容词->En/Ett/Pl词尾变化总结 [Svenska] [Adjectiv]->[En/Ett/Plural/O-/Bestämdform]

Leave a reply

OBS：顺序：En词前 -> Ett词前 -> Plural词前。

 

一、原形 / EN词前 -> ETT单.

1. BRA -> BRA.

bra -> bra

 

2. 元音 -> +TT

t.ex:

ny -> nytt

blå -> blått

 

3.1 ~RD -> ~RT; ~ND -> ~NT

t.ex:

hård -> hårt

känd -> känt

 

3.2 ~D (非RD, ND) -> ~TT

t.ex:

god -> gott

 

4. ~EN -> ~ET

t.ex:
liten -> litet
vaken -> vaket

 

5. ~T -> 不变 (~T)

t.ex:

intressant -> intressant

 

6. 一般 -> +T

t.ex:

dyr -> dyrt

modern -> modernt

fin -> fint

stor -> stort

 

二、原形 / EN词前 -> 复数.

1. BRA -> BRA.

bra -> bra

 

2. LITEN -> SMÅ

liten -> små

 

3. Gammal //辅音前的m不再双写

gammal -> gammalt ->gamla

 

4. 211规则: ~ER, EL, EN

去 e，再按“一般”+a。

t.ex:

vacker -> vackert -> vackra

enkel -> enkelt -> enkla

vaken -> vaket -> vakna

 

4. 一般 -> +A

t.ex:

trevlig -> trevligt -> trevliga

trött -> trött -> trötta

ny -> nytt -> nya

god -> gott -> goda

känd -> känt -> kända

额外变化：结尾m双写

t.ex:

dum -> dumt -> dumma

 

三、Bestämdform.

1. 两个个别: LITEN, BRA

liten 单数 Best-form: lilla, 复数 和 “一般情况”相同.

bra ->bra.

 

2. 一般情况:

不分en/ett, 不分单复数, 都照抄 OBest form 的复数 形式.

//即: 211: 去e, +en.

3. EN ~E(非 ELSE/ARE)/ER/EL/EN/ON ->去E/O, +AR.

t.ex: en pojke ->pojkar, en kalender ->kalendrar, en axel ->axlar, en öken ->öknar, en morgon ->morgnar,

kassar, timmar, påsar; systrar, vintrar, döttrar, fingrar; fåglar, gafflar, reglar /r`e:gel/ ((rule)), stövlar /st`öv:el/ (lång sko); öknar; aftnar.

     3.额外: 有时 O ->Ö: .

t.ex: en dotter ->döttrar,

en moder (=mor) ->mödrar,

 

4. EN ~元 (非 A,E) ->+R.

//因为 AR,ER表示复数, 所以不能直接出现AR,ER, 否则不知道原形, 所以A/E结尾的特殊处理.

t.ex: en sko ->skor,

tår, kor.

 

5. EN ~辅 (非 KER/211) :

5.1 自有词 ->+AR.

t.ex: en bil ->bilar,

tidningar, väggar, stolar, ringar, vägar, båtar, sängar, hundar, bänkar, avdelingar, övningar, meningar, dörrar, film-dukar, räkningar, knivar, skedar, koppar, tallrikar, bussar; armar, pärmar.

    5.1.额外: M,N 发长音时双写.

t.ex: en kam ->kammar,

stammar.

5.2 外来词 ->+ER.

t.ex: en film ->filmer,

studenter, telefoner, parker, texter, poliser, fabriker, familjer, fåtöljer, ingenjörer, servitörer, journalister /ʃor‿nal`is:t/, frisörer, affärer /af`ä:r/, butker, kiosker (liten affär, bara fönster), tid-tab`ell:er, biljetter, cigarretter, sign`a:ler, maskiner /maʃ`i:n/, apels`i:ner, hör-apparater (liten maskin).

    5.2.额外1:  有时 A ->Ä: .

t.ex: händer, tänder, länder, städer, nätter, bokstäver, män.

    5.2.额外2: 有时 O ->Ö: .

t.ex: en bror ->bröder, en son ->söner, en fo:t->f`ött:er, en bok ->böcker.

    5.2.额外3: M,N 发长音时双写.

t.ex: en vän ->vänner.

 

二、ETT

 

1. ETT ~元 (非ELSE) -> +N.

t.ex: ett äpple ->äpplen,

pianon, suddgummin, hjärtan, täcken, foton, kvitton, knän, frimärken, områden, arbeten, ställen, konton, häften, hårstrån, ansikten.

     3.额外: 有俩特例:

ett öga -> ögon, ett öra -> öron.


2. ETT ~辅 -> 不变.

t.ex: ett hus ->hus,

barn, system, exempel, fönster, rum, hem, tak, program, problem, päron, torg, träd, namn, ord.

 

 

Obest复数 ->Best复数

1. EN ~E (//目前只发现~ARE), ETT ~ER/EL ->去E, +NA.

t.ex: lärare ->lärarna, penn-vässare ->penn-vässarna;

exempel ->exemplen, fönster ->fönstren.

 

2. ETT ~辅(单复同型) ->+EN.

t.ex: husen, barnen, systemen.

 

3. 一般: +[N]A (以N结尾的只+A).

t.ex: bilar+na, kvitton+a.

 

4. 额外: M,N发长音时双写.

t.ex: män ->männen.

rummen, hemmen.
